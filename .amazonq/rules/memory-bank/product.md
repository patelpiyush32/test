# Product Overview

## Project Purpose
A hotel SaaS platform built with React and TypeScript, providing hotel management and booking capabilities through a modern web application.

## Key Features
- **Hotel Onboarding**: Multi-step wizard for hotel registration and setup
- **Dashboard**: Comprehensive hotel management interface
- **Website**: Public-facing hotel website for bookings
- **Authentication**: Secure user authentication and authorization
- **Photo Management**: Hotel photo uploads and management
- **Room Management**: Room configuration and pricing
- **Payment Processing**: Integrated payment handling
- **Template System**: Customizable hotel website templates

## Target Users
- Hotel owners and managers
- Hotel staff managing bookings and operations
- Guests booking hotel accommodations

## Use Cases
- Hotel registration and onboarding
- Managing hotel inventory and pricing
- Processing guest bookings
- Managing hotel information and photos
- Viewing booking dashboard and analytics
